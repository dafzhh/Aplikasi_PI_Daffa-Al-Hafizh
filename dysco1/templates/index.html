Color Detection Website On Images

Based Color Detection Web-App with KNN Classifier

How this application work ?


This application accepts user-uploaded image input. When user clicks any part
of the image, the RGB value of the clicked pixel will be saved and given to the
machine learning model pre-train, the model is then receives its value and predicts
the corresponding color name.

How to use this application?
  • Click the "choose file" button.
  • Next you will be directed to the local computer directory, choose the image you want.
  • A new page will appear, and the selected image will appear on the web page.
  • Click anywhere on the image.
  • The color of the pixel that is clicked will appear above the image
  • To review this explanation, refresh the webpage.
